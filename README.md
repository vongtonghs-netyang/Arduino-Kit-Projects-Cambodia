# Arduino Kit Projects Cambodia 🇰🇭

សៀវភៅគម្រោង Arduino ពេញលេញ **៣០ គម្រោង**  
ប្រើសម្ភារទាំង ៩២ ធាតុដែលអ្នកមាន (Arduino UNO R4 WiFi + Expansion Shield + ឧបករណ៍ច្រើនប្រភេទ)

### រចនាសម្ព័ន្ធគម្រោង
- **01_Easy_Level** → គម្រោងទី ១ ដល់ ១០ (កម្រិតងាយស្រួល)
- **02_Medium_Level** → គម្រោងទី ១១ ដល់ ១៨ (កម្រិតធម្យម)
- **03_High_Level** → គម្រោងទី ១៩ ដល់ ៣០ (កម្រិតខ្ពស់)

### តារាងគម្រោងទាំងអស់
| កម្រិត          | គម្រោង | ចំណងជើងគម្រោង                          | សម្ភារសំខាន់ |
|------------------|--------|------------------------------------------|----------------|
| Easy            | 01     | Blink LED                                | LED, Resistor |
| Easy            | 02     | Button Control LED                       | Button, LED   |
| ...             | ...    | ...                                      | ...           |
| High            | 30     | Obstacle Avoidance Robot                 | Ultrasonic, IR, Servo, Motor |

*(នៅពេលបញ្ចប់គម្រោងទាំងអស់ សូមបំពេញតារាងនេះឱ្យពេញ)*

### របៀបប្រើ
1. ចុចលើ folder `01_Easy_Level`, `02_Medium_Level` ឬ `03_High_Level`
2. ចូលទៅគម្រោងណាមួយដែលចង់ប្រើ
3. ទាញយក file `.ino` រួចបើកនៅក្នុង **Arduino IDE**
4. អាន `README.md` នៅក្នុង folder នីមួយៗ ដើម្បីមើលការតភ្ជាប់ និងការពន្យល់

### ធនធានបន្ថែម
- **វីដេអូ Demo** → នឹងមាន QR Code នៅក្នុងសៀវភៅ
- **សម្ភារៈពេញលេញ** → បញ្ជី ៩២ ធាតុ (មាននៅក្នុងសៀវភៅ)
- **User Manual** → ប្រើជាមួយ Paper Book ដែលមានស្រាប់

**សូមស្វាគមន៍មកកាន់គម្រោង Arduino របស់ខ្ញុំ!**  
បើមានសំណួរ សូមបើក **Issue** ឬទាក់ទងតាម Facebook/Telegram។
